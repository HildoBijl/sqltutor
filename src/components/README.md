# Components

Here you find a large variety of general-purpose React components that can be used everywhere on the SQL Valley web app. There are a few subfolders.

- **[html/](./html/)** has a variety of html-like containers. Think of paragraphs, quotes, lists, alerts, code components, etcetera.
- **[flow/](./flow/)** is responsible for managing the flow of the site. It has elements like loading screen and error boundaries.
- **[figures/](./figures/)** contains all functionalities related to rendering and displaying figures on the site. Specifically, it has a custom-made Drawing toolbox to easily render custom figures on pages containing a combination of HTML elements and SVG components.
- **[sql/](./sql/)** has components related to the input, display and running of SQL. This includes implementations of both the CodeMirror SQL input field and the SQL.JS in-browser database.
