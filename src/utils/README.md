# Utility functions

This folder has all general-purpose JavaScript utility functions for the web app. It has the following folders, in order of dependency inclusion.

- [javascript](./javascript/) has functions supporting primitive JavaScript types. If it's a simple function to process your numbers, strings, arrays or objects, you can find it here.
- [geometry](./geometry/) contains classes related to `Vector`, `Line` and `Rectangle` components. Each class has methods providing easy manipulation of the respective shapes.
- [dom](./dom/) has function that support rendering, either through the Document Object Model (DOM) or through React. Any DOM or React support function used in the site comes from this folder.
